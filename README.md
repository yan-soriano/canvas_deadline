# 📅 Canvas Deadline Bot

Telegram-бот для отслеживания дедлайнов, оценок и новых заданий из Canvas LMS.

## ✨ Возможности

- 🆕 **Новые задания** — уведомление при появлении нового задания
- 📊 **Оценки** — уведомление при выставлении или изменении оценки
- ⏰ **Напоминания о дедлайнах** — за 24ч, 12ч, 8ч, 4ч, 2ч и 1ч
- ✅ **Умные уведомления** — если работа уже сдана, напоминания не приходят
- 🎹 **Кнопки меню** — удобная клавиатура в Telegram, не нужно вводить команды
- ⚡ **Параллельные запросы** — быстрая синхронизация с Canvas

## 🚀 Установка

### 1. Клонируй репозиторий
```bash
git clone https://github.com/your-username/canvas_deadline_bot.git
cd canvas_deadline_bot
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Создай `.env` файл
```bash
cp .env.example .env
```

Отредактируй `.env`:
```env
TELEGRAM_TOKEN=твой_токен_от_BotFather
CANVAS_URL=https://canvas.твой_университет.kz/
CHECK_INTERVAL_MINUTES=30
```

### 4. Запусти бота
```bash
python bot.py
```

## 📱 Как пользоваться

1. Открой бота в Telegram
2. Нажми `/start`
3. Создай токен Canvas: **Account → Settings → Approved Integrations → New Access Token**
4. Отправь токен боту
5. Готово! Используй кнопки меню

## ⚙️ Команды

| Команда | Кнопка | Описание |
|---------|--------|----------|
| `/start` | — | Подключение к боту |
| `/deadlines` | 📅 Дедлайны | Список предстоящих дедлайнов |
| `/grades` | 📊 Оценки | Текущие оценки по курсам |
| `/check` | 🔄 Проверить | Принудительная проверка Canvas |
| `/test` | 🧪 Тест | Тестовые уведомления |
| `/settings` | ⚙️ Настройки | Вкл/выкл типы уведомлений |
| `/reconnect` | 🔑 Сменить токен | Смена Canvas токена |
| `/seturl` | — | Сменить URL Canvas |

## 🛡️ Безопасность

- Токен Telegram хранится только в `.env` (не в коде)
- Canvas токен каждого пользователя хранится в локальной БД
- `.env` и `*.db` добавлены в `.gitignore`

## 📋 Требования

- Python 3.10+
- Telegram Bot Token (от [@BotFather](https://t.me/BotFather))
- Canvas LMS аккаунт с API доступом
